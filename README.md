# Confiance Parent POM

Parent POM for all Confiance Platform microservices.

## Usage

In your service `pom.xml`:
```xml
<parent>
    <groupId>com.confiance</groupId>
    <artifactId>confiance-parent</artifactId>
    <version>1.0.0</version>
</parent>
```

## Versions

- Java: 17
- Spring Boot: 3.2.0
- Spring Cloud: 2023.0.0
